### 🏦 Tujijenge v2 — Multi-tenant SACCO SaaS
> *"Let us build ourselves"* — digitalising cooperative finance in Kenya

- 🔐 **PostgreSQL Row-Level Security** for true data isolation between tenants
- 📱 **M-Pesa Daraja API** — STK Push payments, per-tenant credential management  
- 👥 **4 role-based portals** — Member, Organizer, SACCO Admin, Super Admin
- ⚡ **Fastify + Prisma** backend with Redis job queues and background workers
- 🧮 **Credit scoring engine** — reputation score from contribution & repayment history

`TypeScript` `React` `Fastify` `PostgreSQL` `Redis` `Prisma` `M-Pesa` `Docker`
